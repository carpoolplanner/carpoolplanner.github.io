Geocoding via OpenStreetMap Nominatim, driving routes and distances via the public OSRM demo server — both free, rate-limited services best suited to prototyping. For production use with reliable uptime, swap in a licensed provider (Google, Mapbox, HERE). Baseline mileage assumes each rider otherwise drives alone, door to door. Route order per car is chosen to minimize total distance ending at the destination; real-world traffic, one-way streets, and pickup timing aren't modeled. "Plans viewed" counts page loads, not unique people; "Confirmed carpools" is self-reported by whoever clicks the button above.
